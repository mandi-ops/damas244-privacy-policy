# Política de Privacidade — Damas244

**Última atualização: 21 de maio de 2026**

## Introdução

A aplicação Damas244 respeita a sua privacidade. Esta política descreve como recolhemos e utilizamos os seus dados.

## Dados recolhidos

- **Conta**: endereço de e-mail, pseudónimo, avatar
- **Localização**: utilizada apenas para correspondência regional
- **Dados de jogo**: histórico de partidas, estatísticas
- **Publicidade**: identificador publicitário (AdMob/Google)

## Utilização dos dados

Os dados são utilizados para:
- Gerir a sua conta e perfil
- Permitir partidas online em tempo real
- Exibir publicidade relevante (AdMob)
- Melhorar a experiência de jogo

## Partilha de dados

Os dados não são vendidos a terceiros. Partilhamos apenas com:
- **Firebase/Google**: autenticação e base de dados
- **AdMob/Google**: publicidade
